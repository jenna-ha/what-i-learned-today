3. RMSE(Root Mean Square Error) 
    - 잔차의 표준편차 = 잔차가 퍼져있는 정도 = 회귀선을 기준으로 데이터가 얼마나 모여있는지
    - 회귀분석 검증 지표
4. 검증 지표
	- Accuracy = (TP+TN) / (TP+FP+TN+FN)
	- Precision = TP / (TP + FP)
	- Sensitivity/Recall = TP / (TP + FN)
	- Specificity = TN / (FP + TN)
	- F1-score = 2*(Pre * Re) / (Pre + Re)
